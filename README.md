# Assignment 1 – Next.js Rendering Methods

This project demonstrates the different rendering methods in Next.js: SSG, SSR, and CSR.

## Pages / Features
- **SSG (Static Site Generation)**: Pre-rendered pages at build time.
- **SSR (Server Side Rendering)**: Pages rendered on each request.
- **CSR (Client Side Rendering)**: Pages rendered on the client using React hooks.

## Folder Structure
- `pages/` → Contains page components demonstrating SSG, SSR, CSR
- `components/` → Reusable UI components
- `public/` → Static assets
